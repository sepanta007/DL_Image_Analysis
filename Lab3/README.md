# Computer Vision – TP Submissions

Each student must create **one GitHub repository** that will be used for **all Labs during the semester**.

Inside the repository, create **one folder per TP**.

### Required structure:
```text
your-repo/
├── Lab1/
├── Lab2/
├── Lab3/
└── README.md
```

⚠️ Do **not** create a new repository for each TP.
